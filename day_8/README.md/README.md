Day 8 – NumPy and Pandas


What I Learned About NumPy

Today I learned the fundamentals of NumPy, a Python library used for numerical computing and array operations. I started by creating both 1D and 2D arrays and then practiced performing arithmetic operations directly on them. I also learned how to find the maximum, minimum, mean, and sum of array elements, and how to reshape arrays into different dimensions. Indexing and slicing helped me understand how to access specific parts of an array efficiently. Overall, NumPy gave me a clear picture of how numerical data can be stored and processed in a fast and structured way.


What I Learned About Pandas

Pandas is used for working with structured data like CSV files, and today I got hands-on experience with its most important features. I learned how to load a dataset using read_csv() and then explore it using head(), tail(), and info() to understand its structure. I practiced finding missing values, selecting specific rows and columns, and filtering data based on conditions. I also calculated summary statistics using mean() and describe(), created a new column called total_marks, sorted the data based on column values, and finally saved the analyzed dataset into a new CSV file using to_csv().


Key Insights from the Dataset

While working with the Student Performance dataset, I was able to calculate the average marks for each subject and use the total_marks column to identify top-performing students. I filtered out students who scored below the class average, determined the total number of students in the dataset, and saved the final analyzed version as a new CSV file for future use.


Challenges Faced During Implementation

This being my first time working with these libraries, I ran into several beginner-level challenges. Understanding the difference between NumPy arrays and Pandas DataFrames took some time since they look similar but behave differently. Filtering data in Pandas using conditions was a new concept, and I had to practice it a few times before it clicked. I also found it confusing at first that sort_values() sorts the entire row and not just a single column. Calculating averages for specific columns rather than the whole dataset, creating the total_marks column and using it for further analysis, saving a modified DataFrame as a CSV file, and debugging common Pandas errors were all things I had to work through step by step during implementation.


Conclusion

Today's tasks helped me build a solid foundation in both NumPy and Pandas. I now understand how to perform numerical operations using NumPy and how to explore and analyze real-world datasets using Pandas. These skills feel like an important stepping stone toward the Machine Learning tasks coming up in the next few days.
